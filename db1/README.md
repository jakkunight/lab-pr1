# Laboratorio de Programación 1 (PR1): Java y Oracle

Repositorio con las prácticas de laboratorio de PR1 con Java y Oracle.

## Objetivos del laboratorio

- Aprender sobre el manejo de bases de datos relacionales.
- Aprender sobre el desarrollo de aplicaciones con Java.
- Conectar aplicaciones Java con una base de datos Oracle.

## Objetivos personales

- Aplicar buenas práctias de desarrollo de software con Java.
- Realizar el proceso de montaje en NixOS.
